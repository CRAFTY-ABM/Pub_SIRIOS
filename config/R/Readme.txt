Adapt and run machine-specific SIMP file (./R/simp-machine.R) first (see craftyr documentation)!

Then, the output needs to be processed by the process.R script in _setA/_RegGlobMax/cluster/common.
Afterwards, the script for figure creation in the _setA/_RegGlobMax/reports/<RUNS> folder can be executed.
Optionally, processReport.R creates a PDF report for a single run.
